## Introduction

Vue.js uses a syntax similar to HTML that allows you to bind the data in the components to the DOM nodes in the template. All Vue.js templates are valid HTML that can be parsed by spec-compliant browsers and HTML parsers.

Think of what we learned using DOM manipulation. We created the element, added the data, then appended to another element in the DOM. Using Vue.js this is as easy as using its templates

### Directives

You have probably noticed that we used an attribute prefixed with `v-` to interpolate attributes.  This is a common thing in Vue for other functionalities. For example, we use v-if to decide if something is inserted into the HTML. See the example:

```jsx
<p v-if="seen">Now you see me</p>
```

If the component data `seen` is `true`, then this `<p>` HTML element will be inserted in our HTML.

These directives can be more powerful when combined:

- Directives can take **arguments** to be more precise: [https://v3.vuejs.org/guide/template-syntax.html#arguments](https://v3.vuejs.org/guide/template-syntax.html#arguments)
- Directives can take **modifiers** to define its behavior [https://v3.vuejs.org/guide/template-syntax.html#modifiers](https://v3.vuejs.org/guide/template-syntax.html#modifiers)
